# Sheet Fixtures

PDF and image sheet fixtures for deterministic Sheet Library and Sheet Practice viewer tests belong here.

- `real-sheet.pdf`: one-page readable PDF fixture.
- `two-page-sheet.pdf`: two-page readable PDF fixture for page navigation.
- `real-sheet.png`: readable PNG fixture.
- `bad-sheet.pdf` and `bad-sheet.png`: intentionally invalid artifacts for error-state tests.
