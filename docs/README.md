# Documentation

## Current authority

Repository work is routed by [`AGENTS.md`](../AGENTS.md) and native OpenGSD state under [`.planning/`](../.planning/). Current static architecture notes live in [`architecture/`](architecture/).

## Historical material

All pre-OpenGSD product versions, agent indexes, roadmaps, refactor forensics, workflow designs, executed migration plans, and superseded experiments live under [`legacy/`](legacy/). Files below that directory are evidence only: they never authorize current planning or execution, even when their preserved text contains old status labels, commands, or paths.

Current OpenGSD artifacts may link into `docs/legacy/` to preserve provenance. No historical document tree or obsolete workflow entrypoint may exist elsewhere under `docs/`.
