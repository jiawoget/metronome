# Primitive Index (Machine-Readable Repo Map)

This file is generated by `scripts/generate-primitive-index.mjs`.

It is used by coding agents and planners to avoid re-implementing existing logic.

## Rules

Before adding any of the following:

- helper (normalize/format/validate/parse/resolve/select)
- service method
- controller action
- adapter
- browser default injection

You MUST search this index first.

If an existing primitive exists, you MUST reuse it.
If you create a new one, you MUST retire at least one old surface.
